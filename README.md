# Marketing Property Tools

A comprehensive toolkit for real estate professionals to manage and market properties effectively.

## Features

- **Virtual Tours Management**: Create and manage 360° virtual tours
- **Property Analytics**: Track views, engagement, and lead sources
- **Lead Management System**: Capture, track, and manage property inquiries
- **Image Gallery**: Optimize and manage property images
- **Email Campaigns**: Send targeted marketing emails
- **Performance Reports**: Detailed analytics and ROI tracking
- **CRM Integration**: Manage client relationships
- **Social Media Integration**: Post to multiple platforms

## Quick Start

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables (see .env.example)
4. Start the server: `npm start`
